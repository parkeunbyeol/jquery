1. 요소에 이벤트 바인딩

	[1] HTML의 요소에서 이벤트 함수 호출 ==> 최근 권장 자제
		< 요소 onclick='함수명()' >
		
		
	[2] 이벤트 핸들러 : HTML과 JS 분리
		요소변수.onclick = function(){} 
		// 어떤 요소변수 (ex.버튼변수명)이 눌렸을 때 function(){} 즉, 함수가 동작하게 함
		
		
	[3] 이벤트 리스너 : IE8이전 동작하지 않았지만 최근 해결
		요소변수.addEventListener('이벤트명', 함수);


		
		
2. 기본 이벤트 막기--> 방법 2가지인데 걍 외우려하지말고 2가지 다 써보고 막히면 다른 거 쓰기


	[1] preventDefault
			: 원래 이벤트 기능을 막아버림
			[ex] <a>, submit 등등
		
			
	[2] stopPropagation		 	
			: 이벤트를 부모 요소로 버블링(부모한테까지 이벤트가 올라가는 것) 되는 것을 막음
